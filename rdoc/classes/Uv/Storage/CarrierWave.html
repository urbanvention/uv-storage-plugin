Class Uv::Storage::CarrierWave
In: lib/carrierwave/uv_storage.rb
Parent: CarrierWave::Storage::Abstract

Uploads things to urbanvention storage. You‘ll need to specify in the configuration file of urbanvention storage. See Uv::Storage for reference.

You can set the access policy for the uploaded files: 

    CarrierWave.configure do |config|
      config.s3_access_policy = 'public-read'
    end

The default is ‘public-read’. For more options see:

For backwards compatability with amazon s3:

:private
No one else has any access rights.
:public_read
The anonymous principal is granted READ access. If this policy is used on an object, it can be read from a browser with no authentication.
:authenticated_read
Any principal authenticated as a registered Amazon S3 user is granted READ access.

The resulting url will be 

    http://node_name.urbanstorage.com/path/signature
